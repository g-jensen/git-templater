# git-templater

If you want to be able to use the `git-templater graph` command, you need to install [Graph-Easy](https://github.com/ironcamel/Graph-Easy):

### Debian-Based:

```bash
$ sudo apt install libgraph-easy-perl
```
### MacOS:

```bash
$ brew install cpanminus
$ cpan Graph::Easy
```

Everything else should work fine without any extra installation.

## Install

```bash
chmod +x git-templater.sh
```

Then, copy the `git-templater.sh` script onto your path as `git-templater`
